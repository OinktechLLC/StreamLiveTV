КАК ПОДКЛЮЧИТЬ
НА СВОЙ САЙТ
01
Добавьте контейнер
Вставьте div с классом kinoflex-player и атрибутами на странице с фильмом.
02
Подключите скрипт
Один тег <script> с вашим publisher-id — и плеер отрисуется автоматически.
03
Готово
Плеер отображает кнопки источников, сезон/серию для сериалов, VAST-рекламу и сохраняет выбор пользователя.
// Фильм
<!-- 1. Контейнер --> <div class="kinoflex-player" data-kp-id="464963" data-type="movie" data-title="Интерстеллар" data-year="2014" data-height="550"> </div> <!-- 2. Скрипт (один раз на странице) --> <script src="https://kinoflex.ru/embed/player.js" data-publisher-id="ВАШ_ID"> </script>
// Сериал
<div class="kinoflex-player" data-kp-id="77184" data-type="series" data-title="Игра престолов" data-year="2011" data-height="560"> </div> <script src="https://kinoflex.ru/embed/player.js" data-publisher-id="ВАШ_ID"> </script>
// Параметры data-атрибутов
data-kp-id — ID Кинопоиска (обязательно) data-type — "movie" или "series" data-title — название (для поиска VK/YT) data-year — год выхода data-height — высота плеера (px, по умолч. 500)
// демо

KINOFLEX
📦
JS-embed как CDNvideoHub
Один скрипт-тег — плеер рендерится на любом сайте без iframe и без отдельного домена.

Сохранение прогресса
Источник, озвучка, сезон, серия — запоминаются в localStorage для каждого фильма отдельно.

VAST-реклама Clickadilla
Видео-преролл с автопарсингом VAST XML. Кнопка «Пропустить» через 5 сек. Заработок с каждого просмотра.
📋
Навигация по сериям
Автоматически подгружает сезоны и серии с KP API. Скрывается для фильмов, отображается для сериалов.
🔄
Автобалансировка
При ошибке источника — кнопка «Следующий» переключает на следующий в списке автоматически.
