# 📺 SmartTVLove App — Zabawa Wink для РФ

Современный IPTV-плеер с поддержкой M3U/M3U8/HLS/DASH, работающий прямо в браузере.

## Файлы проекта

```
SmartTVLove/
├── index.html        ← Лендинг с туториалом
├── player.html       ← Основной плеер
├── embed.html        ← Embed плеер (для iframe)
├── css/
│   ├── landing.css
│   └── player.css
└── js/
    └── player.js
```

## Как использовать

1. Открой `index.html` в браузере — лендинг с туториалом
2. Нажми «Открыть плеер» → `player.html`
3. Вставь M3U ссылку или нажми «Zabawa Wink»
4. Выбери группу каналов из выпадающего списка
5. Кликни на канал и смотри!

## Забава/Wink плейлисты

Репозиторий: https://github.com/CrocoUser/zabava-project

Откройте репозиторий, найдите .m3u файл, нажмите «Raw» и скопируйте URL.

## Поддерживаемые форматы

- M3U / M3U8
- HLS (HTTP Live Streaming)
- MPEG-DASH
- MP4, WebM, TS, FLV

## Embed код

1. Выберите канал в плеере
2. Нажмите кнопку 🔗 (Embed)
3. Скопируйте iframe код
4. Вставьте на свой сайт — откроется именно выбранный канал

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Пробел | Пауза/воспроизведение |
| ← → | Перемотка ±10 сек |
| ↑ ↓ | Громкость |
| M | Mute |
| F | Полный экран |
